$ curl 使用方法 [展开]

分享文本

echo "hello world" | curl -T - /

分享文件（自动保留文件名）

curl -T file.txt /file.txt

自定义文件名

curl -T data.csv -H 'X-Filename: report.csv' /

设置过期时间

curl -T file.txt -H "X-TTL: 6h" /

wget 上传文件

wget --method=PUT --body-file=file.txt -qO- /
